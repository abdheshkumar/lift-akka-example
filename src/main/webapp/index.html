Welcome to the Liftweb/Akka example project!

This project was presented at the Scala LiftOff Conference 2011 in London. The complete source code can be found at my github repository.

Duplicate a file (on a remote host)

In this example, you can decide which file you want the remote-server to duplicate. It will give you feedback on how much has been copied through a jQuery-UI Dialog with a Progressbar attached to it.

How does it all work?

We are combining the power of Akka with Liftweb's ability to perform Comet-style updates on the client's browser. None of this is scheduled AJAX, it's all Comet. The copy-process in the background uses rsync, so make sure it is in PATH.
